# Assets Directory

This directory contains the image assets for the phone animation section.

## Required Images

Please add the following images to this directory:

1. **iphone.png** - Main iPhone image (centered, will be animated)
2. **case-main.png** - Main case that attaches to the phone during rotation
3. **case-1.png** - Case image for left position
4. **case-2.png** - Case image for right position
5. **case-3.png** - Case image for top position
6. **case-4.png** - Case image for bottom position

## Image Specifications

- **Recommended size for phone**: 300x600px (or similar aspect ratio)
- **Recommended size for cases**: 200x400px (or similar aspect ratio)
- **Format**: PNG with transparent background (preferred)
- **Quality**: High resolution for crisp display
